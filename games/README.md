# Games

Games we track in this repo — sandbox / UGC and **retro emulation** — mostly for **Linux** and **Raspberry Pi**. Each option has its own folder with install notes and links.

- → [Minecraft/](./minecraft/)
    - Paid voxel sandbox: survival, creative, multiplayer, and mods.
    - Free alternatives like [Minecraft Pi: Reborn](https://gitea.thebrokenrail.com/minecraft-pi-reborn/minecraft-pi-reborn) (community revival)
    - Bonus:  **Pi Edition API** for Python block coding ([Streamlit app](./minecraft/app.py)).
- → [Roblox/](./roblox/)
    - Free platform for playing and publishing player-made **experiences**; 
    - **Roblox Studio** for building with Luau. 
- → [Luanti/](./luanti/)
    - Open-source voxel **engine** (formerly Minetest); games and mods from ContentDB, **Lua** scripting.
    - Strong on **Linux** and **Raspberry Pi**; optional home server (no commercial account).
- → [Terasology/](./terasology/)
    - Open-source voxel game with **modules** (Java); **Terasology Launcher** on desktop **Linux x86_64**.
    - Pi client not supported; use Luanti or Minecraft on Pi instead.
- → [Retro/](./retro/)
    - Classic games via **emulation** on Linux and Raspberry Pi.
    - Main focus: **RetroPie** (consoles/handhelds) and **DOSBox** (MS-DOS).

## Other comparable games

No dedicated folder in this repo — listed for comparison and alternatives.

### UGC platforms and online sandboxes

- [Core Games](https://www.coregames.com/) — free UGC platform (Unreal-based); create and play multiplayer worlds on PC
- [Creativerse](https://store.steampowered.com/app/280790/Creativerse/) — Steam voxel sandbox; blueprints, adventures, online worlds
- [Fortnite Creative / UEFN](https://www.fortnite.com/creative) — Epic’s creative mode; different audience than Roblox

### Open source voxel

- [Veloren](https://veloren.net/) — open multiplayer voxel RPG (Rust)
- [Freeminer](https://freeminer.org/) — fork in the Minetest / Luanti family tree

### Play-focused sandboxes (not full UGC platforms)

- [Trove](https://www.trionworlds.com/trove) — MMO voxel adventure
- [Terraria](https://terraria.org/) — 2D sandbox, crafting and bosses

### Learning and game making (not Roblox-style catalogs)

- [Scratch](https://scratch.mit.edu/) — block coding for beginners
- [GDevelop](https://gdevelop.io/) — visual game editor; export to web and mobile
- [Godot](https://godotengine.org/) — open game engine; build your own games (steeper than Studio)

### Small / educational Minecraft-like projects

- [Getting Started with Minecraft Pi](https://projects.raspberrypi.org/en/projects/getting-started-with-minecraft-pi/0) — Raspberry Pi Foundation tutorial
- [StanislavPetrovV/Minecraft](https://github.com/StanislavPetrovV/Minecraft) — Python and OpenGL voxel engine
- [fogleman/Craft](https://github.com/fogleman/Craft) — simple Minecraft-like clone in C and OpenGL
