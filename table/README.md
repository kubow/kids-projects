# Společenské hry

Deskové a karetní hry doma — přehled v jednom souboru na hru; pravidla a odkazy uvnitř.

## Deskové hry

- → [Karak](./karak.md) — průzkum podzemí, 1–4 hráči, kooperace nebo solo
- → [Osadníci z Katanu](./osadnici-z-katanu.md) — obchod a stavba na ostrově, 3–4 hráči
- → [Šachy](./sachy.md) — klasická strategie dvou hráčů
- → [Člověče, nezlob se](./clovece-nezlob-se.md) — dětská hra na kostky, 2–4 hráči

## Karetní hry

- → [Výbušné koťátka](./vybusne-kotatka.md) — rychlá karetní hra, 2–5 hráčů
